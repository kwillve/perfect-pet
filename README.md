## 📘 ***줄거리***
2060년, 원하는 모습으로 반려견을 ‘설계’해 맞춤 제작하는 시대.  
혼자 사는 유나는 어떤 강아지와 함께할지 고민하며  
**‘완벽함’, ‘혼란의 경계’, ‘생명’** 사이의 갈림길에 서게 된다.

> **“완벽한 로봇견, 지하 4층에 마지막으로 남은 진짜 강아지…  
어떤 ‘반려견’과 함께 살아가겠습니까?”**

여러분의 선택에 따라 유나의 미래가 달라집니다!
